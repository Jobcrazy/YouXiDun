# 关于

**网络堡垒(NetGuard)** 是一款专注于游戏行业的 **私有化部署游戏盾** 防护方案，面向高并发、强对抗、低延迟场景，为游戏服务器构建稳定可靠的网络安全防线。产品支持在客户自有服务器或指定机房节点部署，防护链路与数据完全可控，适用于各类在线游戏与实时交互业务。

网络堡垒主打**无限防御**能力，可应对大规模 UDP/TCP 洪水攻击，在高强度攻击环境下依然保障游戏业务稳定运行。同时，系统强调**零误封**策略，通过精细化流量识别与行为判定机制，在清洗攻击流量的同时，最大程度保障正常玩家的访问体验。

网络堡垒的核心源码始终掌握在自身技术团队手中，支持**自建防护节点**与**灵活扩展**，可根据业务规模动态调整防护能力，并提供覆盖 Windows、Android、iOS 的 SDK 接入，满足 APP 与游戏多端集成需求。

结合**源码级维护**与**快速应急响应**，全面支持**抗DDoS**、**抗CC**、**流量加密**等核心业务场景，为游戏厂商提供一套 可控、稳定、可持续演进的游戏盾防护方案，有效抵御各类网络攻击风险。
